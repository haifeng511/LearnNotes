##  TS学习环境搭建

搭建本地TypeScript学习环境

```
下载TypeScript
npm i -g typescript

安装ts-node
npm i -g ts-node

搭建本地环境
tsc --init
这样会创建一个tsconfig.json

新建一个index.ts文件
执行文件代码：
ts-node index.ts
```

在线开发 TypeScript 的云环境：[Playground](https://www.typescriptlang.org/zh/play)
